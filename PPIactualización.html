Actualizaciones

Una actualización de software implica hacer cambios menores en una aplicación o el sistema operativo Android. En estas actualizaciones se incluyen correcciones de errores, parches de seguridad, agregar nuevo soporte para nuevo hardware y otras pequeñas mejoras

BENEFICIOS:

  1. Hacen cambios menores en el software.
  2. Corrigen errores u optimizan el funcionamiento de las funciones que ya tenemos.
  3. Es el software existente, mejorado.
  4. Son rápidas de instalar ya que atienden áreas puntales de los programas o aplicaciones y tienen poco peso.
